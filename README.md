# The Organic Editorial 🛒

A full-stack MERN supermarket application built for the Indian market with secure authentication, cart & order management, responsive UI, and scalable backend architecture.

---

##  Features

- JWT Authentication
- Product Search, Filter & Sorting
- Cart & Wishlist
- Secure Checkout & Orders
- Context API State Management
- Responsive UI with Pure CSS
- REST API Architecture
- MongoDB Integration
- Validation & Security Middleware

---

##  Tech Stack

### Frontend
- React
- React Router DOM
- Context API
- Pure CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Security
- JWT
- bcryptjs
- Helmet
- express-validator
